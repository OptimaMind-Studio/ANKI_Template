# Anki Templates

A collection of Anki card templates for building a personal knowledge dictionary. Record concepts, symbols, theorems, and propositions from various fields (psychology, philosophy, economics, mathematics, investment, CS, AI, etc.) to enhance understanding and build familiarity.

## Available Templates

### Cornell Notes
A concept card template with Cornell Notes layout, featuring AI definitions, personal notes, source references, and English translations.

**Features:**
- AI-generated definitions
- Personal notes section
- Source references (supports Obsidian links)
- Bilingual support (Chinese/English)

**Files:**
- `templates/cornell notes/front.html`
- `templates/cornell notes/back.html`
- `templates/cornell notes/style.css`

**Required Fields:** Name, Field, Taxonomy, References, AIDef, MyNotes

## Installation

1. Open Anki → `Tools` → `Manage Note Types`
2. Select your note type → `Cards...`
3. Copy the HTML and CSS from the template files
4. Ensure all required fields are added to your note type
